# SEO 文章寫手機器人 (Liwei Sia Style)

這個專案根據 `/blog` 資料夾中的內容進行了風格分析，並設計出一套專屬於立崴 (Liwei Sia) 的 SEO 文章寫手機器人架構。

## 特色分析
根據你在 `/blog` 中的現有文章，我們識別出以下風格特徵：
1. **痛點開場**：每篇文章都精確擊中企業主或行銷新手的焦慮。
2. **生活化比喻**：將複雜的廣告邏輯（如外包、Pixel、預算測試）比喻為日常事物。
3. **極高可讀性**：大量使用清單符號與 H2/H3 分層。
4. **誠實建議**：不諱言成本與失敗風險，建立極高的專業信任感。

## 專案結構
- `.agent/prompts/seo_writer.md`: 核心系統提示詞 (System Prompt)，定義了你的語氣、結構與 SEO 規範。
- `writer.ts`: (建議實作) 一個簡單的 TypeScript 腳本，展示如何調用 AI 模型產出文章。

## 如何使用
1. **系統提示詞**：將 `.agent/prompts/seo_writer.md` 中的內容複製到你的 LLM (Gemini, ChatGPT, Claude) 的「系統指令」中。
2. **輸入指令**：
   - 「幫我寫一篇關於 [SEO 關鍵字/主題] 的文章。」
   - 「對象是 [小企業主/行銷新手]。」
   - 「重點要提及 [特定觀點或數據]。」
